Pvjs Diagrammer

Create and Edit Diagrams Online

Install pvjs on your site to create, view and edit diagrams in your browser. We use it for biological pathways, but you can use it for any type of diagram, from flowcharts to electrical circuits.

Customize It

Add your own PNG, JPG, GIF or SVG files as custom shapes. Add your own CSS file to specify custom styling, such as background color or font family.

Integrate External Resources

Enrich your diagram with semantic annotations, connecting your diagram elements to the linked open data cloud. Publish your diagram data as JSON-LD for easy re-use.
