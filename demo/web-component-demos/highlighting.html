Pvjs Web Component Example - Highlighting

The value of the highlights attribute is JSON that has been stringified and URL encoded. For example, the following JSON:

[
  {
    selector: 'ATFS-1',
    backgroundColor: 'yellow',
    borderColor: 'blue',
  },  {
    selector: ['xref:id:http://identifiers.org/wormbase/ZK970.2', 'HAF-1'],
    backgroundColor: 'white',
    borderColor: 'red',
  }
]

becomes:
